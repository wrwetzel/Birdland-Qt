#   This created by hand by consolidating book names from various sources
#   and noralizing / disambiguating some names, particularly the real
#   and real-related books.
#   The canonical name consists of the book name and an optional dash
#   and publisher name. The publisher name is included in the canonical
#   name for some disambiguation but not otherwise used.

#   Format:
#       priority | page_of_sheet_1 | canonical

#   priority - smaller number is higher. Used to select one title
#       among multiple from different canonical books.

#   page_of_sheet_1 can be negative if the first sheet is > 1, e.g.
#       'Jazz Fakebook' where the cover is page 1 but sheet 2 based
#       on subsequent sheet numbering.
#       This is used to generate the consolidated index and offsets.
